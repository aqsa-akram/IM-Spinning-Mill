IM Spinning Mill Management System
A dedicated management and portfolio platform for the textile industry, specifically tailored for spinning mill operations. This project integrates a modern full-stack architecture to handle internal business logic and a professional public presence.

 Tech StackFrontend: 
 React.js (Vite), Tailwind CSS 
 Backend: Node.js, Express.js
 Database: MongoDBTooling: ESLint, Autoprefixer, PostCSS

Key Features:
Textile Portfolio: A professional showcase for the spinning mill's products and services.
Internal Management: Tools for managing mill operations and database records.
Full-Stack Integration: Uses a standardized API base path (/api/v1) for all data operations.
Responsive UI: Built with modern CSS tools to ensure accessibility across devices.
